How probable is each worldview?

Set your subjective credence — your personal degree of belief — that each worldview is correct. Probabilities must sum to 100%; any remainder is unassigned.

What happens to true followers?

For each worldview, set the payoff for someone who sincerely lives according to it — assuming that worldview turns out to be correct. Include any reward and the cost or burden of that way of life.

What happens to followers of other worldviews?

For each worldview, set the outcome for people living according to a different worldview, if this one turns out to be correct. Negative values = penalty; positive = reward or leniency.
