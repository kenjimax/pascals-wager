- Extended-real engine in src/lib/wager.ts must not be modified

8. npm test green, npm run build with GITHUB_PAGES=true produces working static export.

## Surprises and Discoveries
- The parser (oxc via vite) does not allow unescaped apostrophes in string literals the same way, but the real issue was a structural one (presets added outside the array closing bracket)
- Next.js lint rules flag plain img tags even when Next Image optimization is explicitly unwanted for static export
